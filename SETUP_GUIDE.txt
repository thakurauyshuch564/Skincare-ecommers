ISSUE: "Connection failed" error
SOLUTION: 
  - Check if MySQL is running in XAMPP
  - Verify database name is "skincareshop"
  - Check includes/common.php for correct credentials

ISSUE: Admin panel shows blank page
SOLUTION:
  - Clear browser cache
  - Check if session is started
  - Verify admin login credentials

ISSUE: Products not showing images
SOLUTION:
  - Check if images exist in /images/ folder
  - Verify image filenames match database entries

ISSUE: Cart not working
SOLUTION:
  - Ensure user is logged in
  - Check if users_products table exists
  - Verify session is active
